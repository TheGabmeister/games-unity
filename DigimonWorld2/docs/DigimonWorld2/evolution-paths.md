# Digimon World 2 — Complete Digimon Roster & Evolution Paths

195 Digimon total. Data sourced from [MetalKid's DW2 Database](https://github.com/MetalKid/DigimonWorld2_Database).

---

## Key

- **Type:** VAC = Vaccine, DAT = Data, VIR = Virus
- **Element:** Drk = Darkness, Fir = Fire, Mch = Machine, Nat = Nature, Wtr = Water, — = None
- **DP Range:** The Digivolving Points range that triggers this evolution. `0+` means no upper bound.
