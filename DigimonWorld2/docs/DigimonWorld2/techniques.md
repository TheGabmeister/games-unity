## Key

- **Type:** Attack / Interrupt / Counter / Assist
- **Element:** Fire / Water / Nature / Machine / Darkness / None
- **Target:** 1E = One Enemy, AE = All Enemies, RE = Random Enemy, 1A = One Ally, AA = All Allies, Self, All
- **Rank:** Minimum evolution stage to use (Rookie / Champion / Ultimate / Mega)
- **Power:** Base attack power value used in damage formula. Assist skills have no power value.
- **MP:** MP cost per use
