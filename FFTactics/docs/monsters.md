## Beastmaster Bonus Abilities

When a unit with the Beastmaster (Monster Skill) support ability stands adjacent to an allied monster (within 1 tile, height difference ≤ 3), the monster gains access to hidden bonus abilities:

| Monster | Bonus Ability |

## Poaching Drop Table

## Zodiac Compatibility Chart

Every unit has a zodiac sign. Compatibility is determined by the astrological wheel.

### Compatibility by Sign

| Sign | Good (+25%) | Bad (-25%) | Opposite (Best/Worst) |
|---|---|---|---|

| Taurus | Virgo, Capricorn | Leo, Aquarius | Scorpio |
| Gemini | Libra, Aquarius | Virgo, Pisces | Sagittarius |
| Cancer | Scorpio, Pisces | Aries, Libra | Capricorn |
| Leo | Aries, Sagittarius | Taurus, Scorpio | Aquarius |
| Virgo | Taurus, Capricorn | Gemini, Sagittarius | Pisces |
| Libra | Gemini, Aquarius | Cancer, Capricorn | Aries |
| Scorpio | Cancer, Pisces | Leo, Aquarius | Taurus |

| Pisces | Cancer, Scorpio | Gemini, Sagittarius | Virgo |

### Damage Multipliers

| Compatibility | Modifier |
|---|---|
| Best (opposite sign, opposite gender) | ×1.50 |
| Good (trine signs) | ×1.25 |
| Neutral (all other) | ×1.00 |
| Bad (square signs) | ×0.75 |
| Worst (opposite sign, same gender) | ×0.50 |

### Gender Rules

- **Opposite signs:** Best compatibility if opposite gender; Worst if same gender
- **All other sign relationships:** Gender has no effect
- **Monsters** are treated as "opposite gender" to all humans (always Best with compatible opposite signs). Monster-to-monster is always "same gender" (always Worst with opposite signs).
- **Serpentarius** (boss-only zodiac for Elidibus): Neutral with all signs
